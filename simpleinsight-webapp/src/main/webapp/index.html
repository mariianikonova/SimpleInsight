Balanced

More of what you need, less bloat and feature creep.
Even if the whole SimpleInsight experience is built around simplicity it doesn't mean that it is poor in features or lacking in respect to other online learning support systems. We like to think that we managed to build a product that provides everything you need in a simple, functional package.

Cloud powered

It won't go down and it won't loose your data.
SimpleInsight is designed and built to scale and survive. This is not a plain old application running "in the cloud" - the underlying technologies and frameworks that power SimpleInsight have been carefully selected, and its Cloud DNA includes lightweight, portable web components, on top of Linux running on Amazons EC2 and S3.
